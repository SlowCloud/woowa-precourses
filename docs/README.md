# 절차

- 구입 금액을 입력받는다.
- 로또를 발행한다.
- 로또를 출력한다.
- 당첨 번호를 입력받는다.
- 보너스 번호를 입력받는다.
- 당첨 내역을 계산한다.
- 당첨 내역을 출력한다.
- 수익률을 계산한다.
- 수익률을 출력한다.
